Utilizando Rem's Y Em's

El font-size de este parrafo esta en "em's", esto significa que el tamaño de fuente que tiene este parrafo lo hereda del tamaño de fuente que tenga declarado el contenedor padre.
Para la unidad "em" hay que tener en cuenta que tamaño de fuente utiliza su contenedor padre.

El font size de este parrafo esta en "rem's", esto significa que el tamaño de fuente que tiene este parrafo lo hereda del tamaño de fuente que tenga declarado la raiz del documento, puede ser del body o del html.
Para la unidad "rem" hay que tener en cuenta que tamaño de fuente utiliza el documento html.

Este es un parrafo de prueba, y si eres observador puedes notar que el texto tiene sombra...
